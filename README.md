> A lightweight machine learning project that automatically categorizes text documents into predefined classes using the probabilistic principles of Bayes' Theorem (Naive Bayes).

## 📌 Overview
Document classification is a fundamental task in Natural Language Processing (NLP). This project implements a classification model based on **Bayes' Theorem** to automatically assign categories to text documents based on their content. It's designed to be fast, interpretable, and highly effective for text categorization tasks like spam detection, topic labeling, or sentiment analysis.

## ✨ Features
* **Text Preprocessing:** Cleans and prepares raw text data (tokenization, stop-word removal, etc.).
* **Probabilistic Classification:** Utilizes Naive Bayes to calculate the probability of a document belonging to a specific class.
* **Performance Metrics:** Evaluates model accuracy, precision, recall, and F1-score to ensure reliable predictions.

## 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** Scikit-Learn, Pandas, NumPy, NLTK (or standard Python libraries used in your code)
